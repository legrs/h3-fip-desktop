# h3-fip-desktop
conky theme like H3-FIP
H3-FIP(H3ロケット打ち上げのライブ配信画面)ライクなデスクトップ(conkyのconfigファイル)です。

## use
install conky.
`sudo apt install conky-all`

complete config.
- put .conkyrc directory under ~
- put draw.lua directory under ~/.config/conky

start the daemon
`conky -d`

Please edit `.conkyrc` `draw.lua` to change the screen resolution, PC name ,and username.

Other settings can be customized as you like.
